# html-to-markdown

Simple HTML to Markdown parser

# Usage

```js
    const HtmlToMarkdown = require('html-to-markdown');
    const parser = new HtmlToMarkdown();
    const markdown = parser.parse('<h1>Hello World!</h1></br><p>I am a paragraph</p>');
```
